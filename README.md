[English version below | Русская версия ниже]

### 🇷🇺 Описание
Это проект для создания сайта-визитки или пранк-страницы, которую можно записать на NFC-метку. При нажатии на кнопку сайт проигрывает зацикленный звук и перенаправляет пользователя на указанную ссылку.

**Как настроить:**
1. Замените файлы в папке `assets/`:
   - `background.jpg` — ваш фоновый рисунок.
   - `button-icon.png` — ваша картинка-кнопка.
   - `sound.mp3` — аудиофайл (в формате .mp3).
2. Измените URL для редиректа в файле `script.js`.

---

### 🇬🇧 Description
This project is a template for an NFC-triggered landing page or prank site. When the user taps the NFC tag and clicks the button, the site plays a looped sound and redirects the user to a target link.

**How to customize:**
1. Replace the files in the `assets/` folder:
   - `background.jpg` — your background image.
   - `button-icon.png` — your button image.
   - `sound.mp3` — your audio file (in .mp3 format).
2. Change the redirect URL in `script.js`.
